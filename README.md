# hardikkhandhar-sr-fullstack-dev

Professional portfolio website of Hardik Khandhar, Senior Full Stack Developer (9+ years experience).

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript, Glassmorphism design system.
- **Backend**: Node.js, Laravel, PHP, Python & Django, RESTful APIs.
- **Databases & Infrastructure**: MySQL, MongoDB, PostgreSQL, Firebase, Socket.IO, AWS Services.

## Features
- **3-Column Key Achievements Timeline**: Beautiful presentation of professional milestones.
- **3-Column Competencies / Tech Stack**: Clean proficiency indicators matching the main theme.
- **Consolidated Contact Details**: Directly integrated in the profile card.
- **Circular Avatar Crop**: Perfectly centered facial focus.
- **Responsive Layout**: Fluidly optimized for desktop, tablet, and mobile.
